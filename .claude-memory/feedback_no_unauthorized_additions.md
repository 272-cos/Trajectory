Do NOT add features, text, labels, watermarks, or disclaimers that the user did not ask for. When told to replicate a form, replicate it exactly - nothing more. Specifically:
- No "UNOFFICIAL ESTIMATE" watermarks or disclaimers unless explicitly requested
- No extra text or styling beyond what was asked for
- Match the source material exactly; do not embellish

**Why:** User was furious when an unauthorized "UNOFFICIAL ESTIMATE" watermark and red disclaimer were added to the AF Form 4446 PDF replication. They never asked for it and it was added without permission.

**How to apply:** When replicating a form, document, or layout - match it exactly. If you think something should be added (like a disclaimer), ASK first. Do not assume.
